==============================================
   HOME App - PWA (Progressive Web App)
==============================================

الملفات:
- index.html   : الصفحة الرئيسية للتطبيق
- manifest.json: ملف بيانات التطبيق (اسم، أيقونة، ألوان)
- sw.js        : Service Worker للعمل بدون إنترنت والتثبيت
- README.txt   : هذا الملف

طريقة الرفع والاستخدام:
------------------------
1. ارفع الملفات الثلاثة (index.html, manifest.json, sw.js)
   إلى استضافة تدعم HTTPS (شرط أساسي للـ PWA).
   أمثلة: Netlify, Vercel, GitHub Pages, Firebase Hosting,
   أو أي استضافة مع شهادة SSL.

2. تأكد أن الملفات الثلاثة في نفس المجلد.

3. افتح الرابط من متصفح Chrome/Edge/Safari على الهاتف.

طريقة التثبيت من المتصفح:
---------------------------
● Android (Chrome/Edge/Samsung):
  - سيظهر شريط "تثبيت" أسفل الشاشة تلقائياً بعد ثوانٍ.
  - اضغط زر "تثبيت" أو من قائمة المتصفح → "تثبيت التطبيق".
  - سيُثبت كتطبيق كامل بدون شريط عنوان المتصفح.

● iPhone/iPad (Safari):
  - اضغط زر المشاركة ⬆️ في الأسفل.
  - اختر "Add to Home Screen" / "إضافة إلى الشاشة الرئيسية".
  - سيعمل بملء الشاشة كتطبيق حقيقي بدون شريط المتصفح.

● Desktop (Chrome/Edge):
  - أيقونة التثبيت ⊕ في شريط العنوان.

المزايا المضافة:
-----------------
✅ يعمل كتطبيق كامل بدون شريط المتصفح (display:standalone).
✅ زر تثبيت مخصص يظهر تلقائياً.
✅ تعليمات تثبيت خاصة للآيفون.
✅ يعمل بدون إنترنت (offline) بعد أول فتح.
✅ منع التكبير بالضغط المزدوج.
✅ منع القائمة المنبثقة عند الضغط المطول.
✅ يحترم safe-area (النوتش على الآيفون).
✅ أيقونة التطبيق تظهر في شاشة الهاتف الرئيسية.
✅ كل الوظائف الأصلية محفوظة 100%:
   - Firebase Auth + Firestore
   - Recharge / Withdraw / Mining
   - Transactions / Customer Service
   - Language switcher (5 لغات)
   - Theme switcher (4 ثيمات)
   - Tour guide, Gift modal, QR codes
   - Brevo email verification
